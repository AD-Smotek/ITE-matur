# Moderní layout webových stránek

## 1) Co je layout

Layout je **rozložení prvků na webové stránce** – tedy kde se nachází hlavička, menu, obsah, postranní panel a patička.

Moderní layout musí být:
- přehledný
- responzivní (fungovat na mobilu i PC)
- rychlý
- přístupný (accessibility)
- optimalizovaný pro SEO

---

## 2) Tradiční vs. moderní layout

### Starší způsob (tabulky, float)
Dříve se používaly:
- `<table>`
- `float`
- pevné šířky

Nevýhody:
- nepřehledný kód
- složitá údržba
- špatná responzivita

---

### Moderní přístup
Používá:
- sémantické HTML5 prvky
- Flexbox
- CSS Grid
- responzivní design (media queries)

---

## 3) Základní struktura moderní stránky

Typické části:

- `<header>` – hlavička
- `<nav>` – navigace
- `<main>` – hlavní obsah
- `<section>` – sekce obsahu
- `<article>` – samostatný článek
- `<aside>` – postranní panel
- `<footer>` – patička

---

## 4) Layout pomocí Flexboxu

Flexbox se používá pro jednorozměrné rozložení (řádek nebo sloupec).

### HTML

```html
<div class="container">
    <header>Header</header>
    <nav>Menu</nav>
    <main>Obsah</main>
    <aside>Sidebar</aside>
    <footer>Footer</footer>
</div>
```

### CSS

```css
.container {
    display: flex;
    flex-direction: column;
}

nav {
    display: flex;
    justify-content: space-around;
}

main {
    flex: 1;
}
```

Výhody:
- jednoduché centrování
- snadné rozložení prvků
- dobrá podpora pro responzivitu

---

## 5) Layout pomocí CSS Grid

Grid je dvourozměrný systém (řádky i sloupce).

### CSS příklad

```css
.container {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 10px;
}

header {
    grid-column: 1 / 3;
}

footer {
    grid-column: 1 / 3;
}
```

Výhody:
- přesné řízení pozice prvků
- ideální pro složitější layouty
- přehlednější než staré metody

---

## 6) Responzivní design

Moderní web musí fungovat na:
- mobilu
- tabletu
- notebooku
- velkém monitoru

Používají se media queries:

```css
@media (max-width: 768px) {
    .container {
        grid-template-columns: 1fr;
    }
}
```

Dále se používají:
- procenta místo pevných pixelů
- jednotky `rem`, `em`
- flexibilní obrázky (`max-width: 100%`)

---

## 7) Mobile-first přístup

Moderní přístup znamená:
1. Navrhnout design nejdříve pro mobil
2. Postupně přidávat úpravy pro větší obrazovky

Výhody:
- lepší výkon
- lepší UX
- jednodušší údržba

---

## 8) Moderní trendy

- Minimalismus
- Velké nadpisy
- Dostatek bílého místa
- Animace (CSS transitions)
- Dark mode
- Frameworky (Bootstrap, Tailwind)

---

## 9) Shrnutí

Moderní layout webu využívá:
- sémantické HTML prvky
- Flexbox
- CSS Grid
- responzivní design
- mobile-first přístup

Cílem je vytvořit přehledný, rychlý a funkční web, který funguje na všech zařízeních.
