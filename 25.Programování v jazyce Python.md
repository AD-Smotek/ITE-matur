# Programování v jazyce Python

## 1) Co je Python

Python je **vysokoúrovňový, interpretovaný programovací jazyk**.  
Vhodný pro začátečníky i profesionály.  

Vlastnosti:
- čitelná syntaxe
- podpora objektově orientovaného programování
- široká knihovna modulů
- multiplatformní (Windows, Linux, macOS)
- vhodný pro web, vědu, automatizaci, AI a další

---

## 2) Základní struktura programu

```python
# Tento program vypíše "Hello, World!"
print("Hello, World!")
```

- `#` – komentář
- `print()` – výpis na obrazovku

---

## 3) Proměnné a datové typy

### Proměnné
```python
jmeno = "Jan"
vek = 18
vyska = 1.75
```

### Datové typy
- `int` – celé číslo
- `float` – desetinné číslo
- `str` – textový řetězec
- `bool` – logická hodnota (True/False)
- `list` – seznam
- `dict` – slovník
- `tuple` – n-tice

---

## 4) Podmínky

```python
vek = 18

if vek >= 18:
    print("Jste plnoletí")
else:
    print("Jste mladší 18 let")
```

---

## 5) Smyčky

### For
```python
for i in range(1, 6):
    print("Číslo:", i)
```

### While
```python
x = 0
while x < 5:
    print(x)
    x += 1
```

---

## 6) Funkce

```python
def secti(a, b):
    return a + b

vysledek = secti(3, 5)
print("Součet je:", vysledek)
```

- `def` – definice funkce
- `return` – návrat hodnoty

---

## 7) Práce s daty

### Seznamy
```python
seznam = [1, 2, 3, 4]
seznam.append(5)   # přidání prvku
print(seznam[0])   # přístup k prvnímu prvku
```

### Slovníky
```python
student = {"jmeno": "Jan", "vek": 18}
print(student["jmeno"])
```

---

## 8) Import knihoven

Python má mnoho modulů pro různé účely.

```python
import math
print(math.sqrt(16))   # druhá odmocnina

import random
print(random.randint(1, 10))
```

---

## 9) Práce se soubory

```python
# Zápis do souboru
with open("soubor.txt", "w") as f:
    f.write("Ahoj, Python!")

# Čtení ze souboru
with open("soubor.txt", "r") as f:
    obsah = f.read()
    print(obsah)
```

## 10) Shrnutí

Python je jednoduchý a čitelný jazyk.  
Umožňuje:

- práci s daty
- tvorbu funkcí a modulů
- podmínky a smyčky
- interakci se soubory
- použití knihoven pro matematiku, náhodu, web, vědu a AI

Python je ideální pro začátečníky i profesionály díky své univerzálnosti a čitelnosti.
