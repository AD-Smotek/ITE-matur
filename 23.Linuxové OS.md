# Linuxové operační systémy

## 1) Co je Linux

Linux je **open-source operační systém** založený na Unixu.  
V jádru Linux spravuje hardware a umožňuje běh programů.  

Výhody:
- zdarma
- otevřený zdrojový kód
- bezpečný a stabilní
- široké možnosti přizpůsobení

---

## 2) Historie

- 1991 – Linus Torvalds vytvořil první jádro Linuxu  
- Inspirován Unixem  
- Díky open-source komunitě vznikly stovky distribucí

---

## 3) Distribuce Linuxu

---

## 4) Složky Linuxu

Linuxový OS se skládá z:

- **Jádro (Kernel)** – komunikuje s hardwarem  
- **Shell** – příkazové rozhraní  
- **Soubory a adresáře** – organizace dat  
- **Programy a služby** – desktop, server, utilitky  

---

## 5) Práce s příkazovou řádkou

Základní příkazy:

| Příkaz | Funkce |
|--------|--------|
| `ls` | výpis souborů |
| `cd` | změna adresáře |
| `pwd` | aktuální adresář |
| `mkdir` | vytvoření složky |
| `rm` | odstranění souboru/složky |
| `cp` | kopírování |
| `mv` | přesun/rename |
| `chmod` | změna oprávnění |
| `sudo` | spuštění s administrátorskými právy |

---

## 6) Správa software

- `apt` (Debian/Ubuntu) – instalace balíčků  
  ```bash
  sudo apt update
  sudo apt install nazov_programu
  ```
- `yum` nebo `dnf` (Fedora, CentOS)  
- `pacman` (Arch Linux)

---

## 7) Systém práv a uživatelů

Linux je víceuživatelský OS. Každý uživatel má:

- **UID (User ID)**  
- Domácí adresář  
- Přístupová práva ke souborům (rwx – read, write, execute)  

Root uživatel = administrátor, může měnit vše.

---

## 8) Procesy a služby

- `ps` – výpis běžících procesů  
- `top` – živý přehled procesů  
- `systemctl` – správa služeb (start, stop, restart)  

---

## 9) Výhody Linuxu

- Stabilní a spolehlivý  
- Bezpečný, méně náchylný k virům  
- Přizpůsobitelný pro různé účely (server, desktop, embedded)  
- Open-source komunita a podpora  

---

## 10) Shrnutí

Linux je flexibilní, bezpečný a stabilní operační systém vhodný pro:

- servery
- desktopové počítače
- vývoj
- embedded zařízení

Díky otevřenému kódu je široce používaný a neustále vylepšován komunitou.
