# Webové formuláře a práce s daty

## 1) Co je webový formulář

Webový formulář slouží k **zadávání a odesílání dat uživatelem**.  
Používá se například pro:

- registraci
- přihlášení
- objednávky
- kontaktní formuláře
- vyhledávání

Formuláře jsou vytvářeny pomocí HTML značky `<form>`.

---

## 2) Základní struktura formuláře

```html
<form action="zpracovani.php" method="post">
    
    <label for="jmeno">Jméno:</label>
    <input type="text" id="jmeno" name="jmeno">

    <br><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <br><br>

    <input type="submit" value="Odeslat">

</form>
```

### Vysvětlení:

- `action` – kam se data odešlou
- `method` – způsob odeslání dat (GET nebo POST)
- `label` – popisek pole
- `input` – vstupní pole
- `submit` – tlačítko pro odeslání

---

## 3) Metody odesílání dat

### GET
- Data jsou viditelná v URL
- Omezená délka
- Používá se pro vyhledávání
- Méně bezpečné

Příklad URL:
```
stranka.php?jmeno=Jan
```

---

### POST
- Data nejsou vidět v URL
- Větší množství dat
- Používá se pro přihlášení, registraci
- Bezpečnější než GET

---

## 4) Typy vstupních polí

### Text
```html
<input type="text">
```

### Heslo
```html
<input type="password">
```

### Email
```html
<input type="email">
```

### Číslo
```html
<input type="number">
```

### Datum
```html
<input type="date">
```

### Zaškrtávací pole (checkbox)
```html
<input type="checkbox">
```

### Přepínač (radio)
```html
<input type="radio">
```

### Výběrové menu
```html

    <option>Možnost 1</option>
    <option>Možnost 2</option>
</select>
```

### Textová oblast
```html
<textarea></textarea>
```

---

## 5) Validace formulářů

Validace kontroluje správnost dat.

### HTML validace

```html
<input type="email" required>
```

Atributy:
- `required` – povinné pole
- `minlength`, `maxlength`
- `pattern` – regulární výraz
- `min`, `max`

---

### JavaScript validace

Používá se pro pokročilejší kontrolu:
- kontrola shody hesel
- kontrola formátu
- dynamické chyby

---

## 6) Práce s daty na serveru

Po odeslání formuláře jsou data zpracována na serveru (např. pomocí PHP, Node.js, Python).

Příklad v PHP:

```php
<?php
$jmeno = $_POST["jmeno"];
echo "Ahoj " . $jmeno;
?>
```

Data lze:
- uložit do databáze
- poslat emailem
- zpracovat pro objednávku

---

## 7) Bezpečnost formulářů

Důležité zásady:

- Ověřovat data na serveru
- Používat HTTPS
- Ochrana proti SQL injection
- Ochrana proti XSS
- Hashování hesel

Nikdy nedůvěřovat datům od uživatele.

---

## 8) Shrnutí

Webové formuláře slouží ke sběru dat od uživatele.  
Používají značku `<form>` a různé typy vstupních polí.  
Data se odesílají metodou GET nebo POST.  
Musí se validovat a bezpečně zpracovat na serveru.

Formuláře jsou klíčovou součástí interaktivních webových aplikací.
