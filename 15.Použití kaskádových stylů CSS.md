# Použití kaskádových stylů (CSS)

## 1) Co je CSS

CSS (Cascading Style Sheets) je jazyk určený pro **úpravu vzhledu webových stránek**.  
Používá se společně s HTML.

HTML = struktura  
CSS = vzhled (barvy, velikosti, rozložení)  
JavaScript = funkčnost

CSS umožňuje nastavit:
- barvy textu a pozadí
- velikost a typ písma
- okraje a zarovnání
- rozložení prvků na stránce
- responzivní design

---

## 2) Způsoby připojení CSS k HTML

### a) Inline styl (přímo v tagu)

```html
<p style="color: red;">Červený text</p>
```

Nevýhoda: nepřehledné, nepoužívá se ve větších projektech.

---

### b) Interní styl (v hlavičce dokumentu)

```html
<head>
    <style>
        p {
            color: blue;
        }
    </style>
</head>
```

Platí pouze pro danou stránku.

---

### c) Externí styl (doporučený způsob)

HTML:
```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

Soubor `style.css`:
```css
p {
    color: green;
}
```

Výhody:
- přehlednost
- oddělení struktury a vzhledu
- možnost použití na více stránkách

---

## 3) Základní syntaxe CSS

CSS se skládá z:

```
selektor {
    vlastnost: hodnota;
}
```

Příklad:
```css
h1 {
    color: blue;
    font-size: 30px;
}
```

- Selektor – určuje, na co se styl aplikuje
- Vlastnost – co měníme
- Hodnota – jak to měníme

---

## 4) Selektory

### 1) Podle názvu tagu
```css
p {
    color: red;
}
```

### 2) Třída (class)

HTML:
```html
<p class="highlight">Text</p>
```

CSS:
```css
.highlight {
    background-color: yellow;
}
```

Používá se tečka `.`

---

### 3) ID

HTML:
```html
<p id="hlavni">Text</p>
```

CSS:
```css
#hlavni {
    color: blue;
}
```

Používá se mřížka `#`

ID by mělo být na stránce jen jednou.

---

### 4) Potomek (zanoření)

```css
div p {
    color: green;
}
```

Platí jen pro `<p>` uvnitř `<div>`.

---

## 5) Kaskáda a priorita

CSS znamená „kaskádové“ – styly se dědí a mají různou prioritu.

Priorita:
1. Inline styl
2. ID
3. Třída
4. Tag

Při stejné prioritě rozhoduje pořadí v souboru (poslední vyhrává).

---

## 6) Nejčastější vlastnosti

### Barvy
```css
color: red;
background-color: yellow;
```

### Text
```css
font-size: 16px;
font-family: Arial;
text-align: center;
```

### Box model

Každý prvek je obdélník:

```css
width: 200px;
height: 100px;
margin: 10px;
padding: 15px;
border: 1px solid black;
```

Box model obsahuje:
- content (obsah)
- padding (vnitřní okraj)
- border (ohraničení)
- margin (vnější okraj)

---

## 7) Rozložení prvků

### Display

```css
display: block;
display: inline;
display: inline-block;
```

### Flexbox (moderní rozložení)

```css
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
```

Používá se pro jednoduché a responzivní rozložení.

---

## 8) Responzivní design

Umožňuje přizpůsobení stránky mobilům.

```css
@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}
```

---

## 9) Shrnutí

CSS slouží k úpravě vzhledu webových stránek.  
Odděluje obsah (HTML) od designu.  
Používá selektory, vlastnosti a hodnoty.  
Funguje na principu kaskády a priority.  
Umožňuje moderní rozložení pomocí Flexboxu a responzivní design.

Bez CSS by webové stránky byly pouze černý text na bílé stránce.
