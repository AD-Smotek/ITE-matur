# Ukázka skriptu pro Linux OS

## 1) Co je skript

Skript je **soubor s příkazy**, které OS vykoná postupně.  
V Linuxu se často používají **bash skripty** (.sh) pro automatizaci úloh.

Výhody:
- automatizace opakujících se úloh
- správa systému
- zálohy, instalace, monitoring

---

## 2) Základní struktura bash skriptu

```bash
#!/bin/bash
# Tento skript vypíše "Hello, World!" a informace o systému

echo "Hello, World!"

# Informace o uživateli
echo "Jste přihlášeni jako: $USER"

# Datum a čas
echo "Dnes je: $(date)"

# Volné místo na disku
echo "Volné místo na disku:"
df -h

# Seznam souborů v domovském adresáři
echo "Soubory ve vašem domovském adresáři:"
ls ~
```

### Vysvětlení:

- `#!/bin/bash` – shebang, určuje interpreter
- `echo` – výpis textu
- `$USER` – proměnná s aktuálním uživatelem
- `$(date)` – příkaz, jehož výstup se vloží
- `df -h` – přehled volného místa na discích
- `ls ~` – výpis souborů domovského adresáře

---

## 3) Spuštění skriptu

1. Udělat skript spustitelným:

```bash
chmod +x muj_skript.sh
```

2. Spustit skript:

```bash
./muj_skript.sh
```

---

## 4) Přidání parametrů

Parametry se zadávají při spuštění:

```bash
#!/bin/bash
echo "První parametr: $1"
echo "Druhý parametr: $2"
```

Spuštění:

```bash
./muj_skript.sh Jan 18
```

Výstup:
```
První parametr: Jan
Druhý parametr: 18
```

---

## 5) Podmínky a cykly

### Podmínka

```bash
#!/bin/bash
if [ -f "soubor.txt" ]; then
    echo "Soubor existuje."
else
    echo "Soubor neexistuje."
fi
```

### Cyklus for

```bash
#!/bin/bash
for i in 1 2 3 4 5
do
    echo "Číslo: $i"
done
```

---

## 6) Shrnutí

Bash skript umožňuje:

- automatizovat příkazy
- pracovat s proměnnými a parametry
- používat podmínky a cykly
- získat informace o systému (uživatel, disk, soubory)

Skripty jsou základní nástroj správců Linuxových systémů a pro automatizaci úloh.
