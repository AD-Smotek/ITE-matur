# Virtualizace a cloud computing

## 1) Co je virtualizace

Virtualizace je technologie, která umožňuje **běh více operačních systémů nebo prostředí na jednom fyzickém počítači**.  

Cíle virtualizace:
- efektivní využití hardwaru
- izolace systémů
- snadná správa
- testování a vývoj

---

## 2) Typy virtualizace

### 1) Plná virtualizace
- virtuální stroj má vlastní operační systém
- host OS se chová jako samostatný počítač
- příklad: VMware, VirtualBox

### 2) Paravirtualizace
- host a guest OS spolupracují
- vyšší výkon
- méně kompatibility

### 3) Virtualizace na úrovni OS (containerization)
- společné jádro OS
- izolované aplikace v kontejnerech
- příklad: Docker, LXC

---

## 3) Výhody virtualizace

- efektivní využití zdrojů
- snazší zálohování a migrace
- testování různých OS bez nového hardwaru
- izolace aplikací (bezpečnost)
- snadná správa serverů

---

## 4) Cloud computing

Cloud computing je **poskytování výpočetních zdrojů přes internet**.  
Uživatelé nemusí spravovat hardware, vše běží v datacentru poskytovatele.

Typy služeb:

### 1) IaaS (Infrastructure as a Service)
- poskytuje virtuální servery, úložiště, sítě
- příklad: Amazon EC2, Microsoft Azure

### 2) PaaS (Platform as a Service)
- poskytuje platformu pro vývoj aplikací
- uživatel nemusí spravovat OS ani middleware
- příklad: Google App Engine, Heroku

### 3) SaaS (Software as a Service)
- kompletní aplikace přes web
- uživatel jen používá software
- příklad: Gmail, Office 365

---

## 5) Výhody cloud computingu

- nižší náklady na hardware
- flexibilní škálování (scale up/down)
- dostupnost odkudkoli
- automatické zálohování
- správa a bezpečnost zajišťuje poskytovatel

---

## 6) Virtualizace a cloud computing dohromady

- Cloud využívá virtualizaci pro běh více virtuálních strojů na jednom fyzickém serveru
- Kontejnery umožňují rychlé nasazení aplikací v cloudu
- Virtualizace zvyšuje efektivitu, cloud zajišťuje dostupnost a škálovatelnost

---

## 7) Shrnutí

- Virtualizace umožňuje spouštět více systémů na jednom fyzickém hardware.
- Cloud computing poskytuje výpočetní zdroje a aplikace přes internet.
- Obě technologie šetří náklady, zvyšují flexibilitu a usnadňují správu IT infrastruktury.
